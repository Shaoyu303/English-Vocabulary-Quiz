# 英文單字自我測驗程式 (English Vocabulary Quiz)
這是我製作的一個 C++ 練習專案，旨在透過互動式命令列介面，幫助使用者練習核心英文單字。
## 專案亮點
* **隨機出題**：使用 `srand` 與 `rand` 實作隨機選題機制。
* **互動邏輯**：具備三次錯誤嘗試機會，提升使用者體驗。
* **資料結構**：運用陣列（Array）管理單字與對應中文翻譯。
## 學習心得
在開發這個小工具的過程中，我學習到了如何處理 C++ 的字串比較、迴圈控制以及隨機變數的運
用。未來我計畫加入「外部檔案讀取」功能，讓單字庫可以無限擴張。
## 如何執行
1. 使用任何 C++ 編譯器（如 Dev-C++ 或 VS Code）。
2. 編譯並執行 `main.cpp` 即可開始測驗。
